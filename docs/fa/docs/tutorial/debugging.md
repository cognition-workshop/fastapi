# اشکال‌زدایی

شما می‌توانید دیباگر را در ویرایشگر خود متصل کنید، برای مثال با Visual Studio Code یا PyCharm.

## فراخوانی `uvicorn`

در برنامه FastAPI خود، `uvicorn` را وارد و مستقیماً اجرا کنید:

{* ../../docs_src/debugging/tutorial001.py hl[1,15] *}

### درباره `__name__ == "__main__"`

هدف اصلی `__name__ == "__main__"` این است که کدی داشته باشید که وقتی فایل شما به این صورت فراخوانی می‌شود اجرا شود:

<div class="termy">

```console
$ python myapp.py
```

</div>

اما وقتی فایل دیگری آن را وارد می‌کند، فراخوانی نشود، مانند:

```Python
from myapp import app
```

#### جزئیات بیشتر

فرض کنید فایل شما `myapp.py` نام دارد.

اگر آن را با این دستور اجرا کنید:

<div class="termy">

```console
$ python myapp.py
```

</div>

آنگاه متغیر داخلی `__name__` در فایل شما، که به طور خودکار توسط پایتون ایجاد شده، مقدار رشته `"__main__"` را خواهد داشت.

بنابراین، بخش:

```Python
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

اجرا خواهد شد.

---

اگر آن ماژول (فایل) را وارد کنید این اتفاق نخواهد افتاد.

بنابراین، اگر فایل دیگری `importer.py` با این محتوا داشته باشید:

```Python
from myapp import app

# کد بیشتر
```

در آن صورت، متغیر ایجاد شده خودکار درون `myapp.py` متغیر `__name__` با مقدار `"__main__"` نخواهد داشت.

بنابراین، خط:

```Python
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

اجرا نخواهد شد.

/// info

برای اطلاعات بیشتر، <a href="https://docs.python.org/3/library/__main__.html" class="external-link" target="_blank">مستندات رسمی پایتون</a> را بررسی کنید.

## اجرای کد با دیباگر

چون سرور Uvicorn را مستقیماً از کد خود اجرا می‌کنید، می‌توانید برنامه پایتون خود (برنامه FastAPI) را مستقیماً از دیباگر فراخوانی کنید.

---

برای مثال، در Visual Studio Code می‌توانید:

* به پنل "Debug" بروید.
* "Add configuration..." را بزنید.
* "Python" را انتخاب کنید.
* دیباگر را با گزینه "`Python: Current File (Integrated Terminal)`" اجرا کنید.

سپس سرور با کد **FastAPI** شما شروع به کار می‌کند، در نقاط شکست شما متوقف می‌شود و غیره.

اگر از Pycharm استفاده می‌کنید، می‌توانید:

* منوی "Run" را باز کنید.
* گزینه "Debug..." را انتخاب کنید.
* سپس یک منوی زمینه ظاهر می‌شود.
* فایل مورد نظر برای اشکال‌زدایی را انتخاب کنید (در این مورد، `main.py`).
