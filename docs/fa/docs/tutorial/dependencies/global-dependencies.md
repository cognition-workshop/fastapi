بعداً، هنگام مطالعه درباره نحوه ساختاردهی برنامه‌های بزرگ‌تر ([برنامه‌های بزرگ‌تر - فایل‌های متعدد](../../tutorial/bigger-applications.md){.internal-link target=_blank})، احتمالاً با فایل‌های متعدد، یاد خواهید گرفت که چگونه یک پارامتر `dependencies` واحد را برای گروهی از *عملیات‌های مسیر* تعریف کنید.
