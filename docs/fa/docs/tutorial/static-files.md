# فایل‌های استاتیک

شما می‌توانید فایل‌های استاتیک را به طور خودکار از یک دایرکتوری با استفاده از `StaticFiles` سرو کنید.

## استفاده از `StaticFiles`

* `StaticFiles` را وارد کنید.
* یک نمونه `StaticFiles()` را در یک مسیر خاص "سوار" کنید.

{* ../../docs_src/static_files/tutorial001.py hl[2,6] *}

/// note | جزئیات فنی

شما همچنین می‌توانید از `from starlette.staticfiles import StaticFiles` استفاده کنید.

**FastAPI** همان `starlette.staticfiles` را به عنوان `fastapi.staticfiles` فقط به عنوان یک سهولت برای شما به عنوان برنامه‌نویس فراهم می‌کند. اما در واقع مستقیماً از Starlette می‌آید.

///

### "سوار کردن" چیست

"سوار کردن" به معنای افزودن یک برنامه کاملاً "مستقل" در یک مسیر خاص است که سپس مسئولیت مدیریت تمام زیرمسیرها را بر عهده می‌گیرد.

این با استفاده از `APIRouter` متفاوت است زیرا یک برنامه سوار شده کاملاً مستقل است. OpenAPI و مستندات برنامه اصلی شما شامل هیچ چیزی از برنامه سوار شده نخواهد بود و غیره.

می‌توانید درباره این موضوع در [راهنمای کاربر پیشرفته](../advanced/index.md){.internal-link target=_blank} بیشتر بخوانید.

## جزئیات

اولین `"/static"` به زیرمسیری اشاره دارد که این "زیربرنامه" روی آن "سوار" خواهد شد. بنابراین، هر مسیری که با `"/static"` شروع شود توسط آن مدیریت خواهد شد.

`directory="static"` به نام دایرکتوری حاوی فایل‌های استاتیک شما اشاره دارد.

`name="static"` یک نام به آن می‌دهد که می‌تواند به صورت داخلی توسط **FastAPI** استفاده شود.

تمام این پارامترها می‌توانند متفاوت از "`static`" باشند، آنها را با نیازها و جزئیات خاص برنامه خود تنظیم کنید.
