# رمزگذار سازگار با JSON

مواردی وجود دارد که ممکن است نیاز به تبدیل یک نوع داده (مانند مدل Pydantic) به چیزی سازگار با JSON (مانند `dict`، `list` و غیره) داشته باشید.

برای مثال، اگر نیاز به ذخیره آن در پایگاه داده دارید.

برای این کار، **FastAPI** تابع `jsonable_encoder()` را فراهم می‌کند.

## استفاده از `jsonable_encoder`

تصور کنید که یک پایگاه داده `fake_db` دارید که فقط داده‌های سازگار با JSON را دریافت می‌کند.

برای مثال، اشیاء `datetime` را دریافت نمی‌کند، زیرا آنها با JSON سازگار نیستند.

بنابراین، یک شیء `datetime` باید به یک `str` حاوی داده در <a href="https://en.wikipedia.org/wiki/ISO_8601" class="external-link" target="_blank">فرمت ISO</a> تبدیل شود.

به همین ترتیب، این پایگاه داده یک مدل Pydantic (یک شیء با ویژگی‌ها) را دریافت نمی‌کند، فقط یک `dict`.

می‌توانید از `jsonable_encoder` برای این کار استفاده کنید.

یک شیء مانند مدل Pydantic را دریافت می‌کند و نسخه سازگار با JSON آن را برمی‌گرداند:

{* ../../docs_src/encoder/tutorial001_py310.py hl[4,21] *}

در این مثال، مدل Pydantic را به یک `dict` و `datetime` را به یک `str` تبدیل می‌کند.

نتیجه فراخوانی آن چیزی است که می‌تواند با <a href="https://docs.python.org/3/library/json.html#json.dumps" class="external-link" target="_blank">`json.dumps()`</a> استاندارد پایتون رمزگذاری شود.

یک `str` بزرگ حاوی داده در فرمت JSON (به عنوان رشته) برنمی‌گرداند. یک ساختار داده استاندارد پایتون (مثلاً یک `dict`) با مقادیر و زیرمقادیری که همگی با JSON سازگار هستند برمی‌گرداند.
