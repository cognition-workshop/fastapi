# فرم‌های درخواست و فایل‌ها

شما می‌توانید فایل‌ها و فیلدهای فرم را به طور همزمان با استفاده از `File` و `Form` تعریف کنید.

/// info

برای دریافت فایل‌های آپلود شده و/یا داده‌های فرم، ابتدا <a href="https://github.com/Kludex/python-multipart" class="external-link" target="_blank">`python-multipart`</a> را نصب کنید.

مطمئن شوید که یک [محیط مجازی](../virtual-environments.md){.internal-link target=_blank} ایجاد کرده‌اید، آن را فعال کنید و سپس نصب کنید، برای مثال:

```console
$ pip install python-multipart
```

## تعریف پارامترهای `File` و `Form`

پارامترهای فایل و فرم را به همان روشی که برای `Body` یا `Query` انجام می‌دهید ایجاد کنید:

{* ../../docs_src/request_forms_and_files/tutorial001_an_py39.py hl[10:12] *}

فایل‌ها و فیلدهای فرم به عنوان داده‌های فرم آپلود خواهند شد و شما فایل‌ها و فیلدهای فرم را دریافت خواهید کرد.

و می‌توانید برخی از فایل‌ها را به عنوان `bytes` و برخی را به عنوان `UploadFile` تعریف کنید.

/// warning

شما می‌توانید چندین پارامتر `File` و `Form` را در یک *عملیات مسیر* تعریف کنید، اما نمی‌توانید فیلدهای `Body` را که انتظار دارید به عنوان JSON دریافت شوند نیز تعریف کنید، زیرا بدنه درخواست به جای `application/json` با `multipart/form-data` کدگذاری خواهد شد.

این محدودیتی از **FastAPI** نیست، بلکه بخشی از پروتکل HTTP است.

///

## خلاصه

هنگامی که نیاز به دریافت داده‌ها و فایل‌ها در یک درخواست دارید، از `File` و `Form` با هم استفاده کنید.
