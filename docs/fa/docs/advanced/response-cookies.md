# کوکی‌های پاسخ

## استفاده از پارامتر `Response`

شما می‌توانید یک پارامتر از نوع `Response` را در *تابع عملیات مسیر* خود تعریف کنید.

و سپس می‌توانید کوکی‌ها را در آن شیء پاسخ *موقت* تنظیم کنید.

{* ../../docs_src/response_cookies/tutorial002.py hl[1, 8:9] *}

و سپس می‌توانید هر شیئی که نیاز دارید را مانند حالت عادی برگردانید (یک `dict`، یک مدل پایگاه داده و غیره).

و اگر `response_model` تعریف کرده باشید، همچنان برای فیلتر و تبدیل شیئی که برگردانده‌اید استفاده خواهد شد.

**FastAPI** از آن پاسخ *موقت* برای استخراج کوکی‌ها (همچنین هدرها و کد وضعیت) استفاده خواهد کرد و آنها را در پاسخ نهایی که شامل مقداری است که برگردانده‌اید و توسط هر `response_model` فیلتر شده، قرار خواهد داد.

همچنین می‌توانید پارامتر `Response` را در وابستگی‌ها تعریف کنید و کوکی‌ها (و هدرها) را در آنها تنظیم کنید.

## برگرداندن مستقیم `Response`

همچنین می‌توانید هنگام برگرداندن مستقیم یک `Response` در کد خود، کوکی‌ها را ایجاد کنید.

برای این کار، می‌توانید یک پاسخ را همانطور که در [برگرداندن مستقیم پاسخ](response-directly.md){.internal-link target=_blank} توضیح داده شده ایجاد کنید.

سپس کوکی‌ها را در آن تنظیم کنید و آن را برگردانید:

{* ../../docs_src/response_cookies/tutorial001.py hl[10:12] *}

/// tip

به خاطر داشته باشید که اگر یک پاسخ را مستقیماً به جای استفاده از پارامتر `Response` برگردانید، FastAPI آن را مستقیماً برمی‌گرداند.

بنابراین، باید مطمئن شوید که داده‌های شما از نوع صحیح هستند. مثلاً اگر از `JSONResponse` استفاده می‌کنید، با JSON سازگار باشند.

و همچنین داده‌ای که باید توسط `response_model` فیلتر می‌شد ارسال نکنید.

### اطلاعات بیشتر

/// note | جزئیات فنی

شما همچنین می‌توانید از `from starlette.responses import Response` یا `from starlette.responses import JSONResponse` استفاده کنید.

**FastAPI** همان `starlette.responses` را به عنوان `fastapi.responses` فقط به عنوان یک سهولت برای شما به عنوان برنامه‌نویس فراهم می‌کند. اما بیشتر پاسخ‌های موجود مستقیماً از Starlette می‌آیند.

و از آنجا که `Response` می‌تواند به طور مکرر برای تنظیم هدرها و کوکی‌ها استفاده شود، **FastAPI** آن را در `fastapi.Response` نیز فراهم می‌کند.
