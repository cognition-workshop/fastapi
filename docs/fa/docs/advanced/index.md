# راهنمای کاربر پیشرفته

## ویژگی‌های اضافی

[آموزش - راهنمای کاربر](../tutorial/index.md){.internal-link target=_blank} اصلی باید برای معرفی تمام ویژگی‌های اصلی **FastAPI** کافی باشد.

در بخش‌های بعدی، گزینه‌ها، تنظیمات و ویژگی‌های اضافی دیگری را خواهید دید.

/// tip

بخش‌های بعدی **لزوماً "پیشرفته" نیستند**.

و ممکن است برای مورد استفاده شما، راه‌حل در یکی از آنها باشد.

///

## ابتدا آموزش را بخوانید

شما همچنان می‌توانید بیشتر ویژگی‌های **FastAPI** را با دانشی که از [آموزش - راهنمای کاربر](../tutorial/index.md){.internal-link target=_blank} اصلی کسب کرده‌اید استفاده کنید.

و بخش‌های بعدی فرض می‌کنند که شما قبلاً آن را خوانده‌اید و آن ایده‌های اصلی را می‌دانید.
