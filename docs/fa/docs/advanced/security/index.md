# امنیت پیشرفته

## ویژگی‌های اضافی

ویژگی‌های اضافی برای مدیریت امنیت علاوه بر مواردی که در [آموزش - راهنمای کاربر: امنیت](../../tutorial/security/index.md){.internal-link target=_blank} پوشش داده شده‌اند وجود دارد.

/// tip

بخش‌های بعدی **لزوماً "پیشرفته" نیستند**.

و ممکن است برای مورد استفاده شما، راه‌حل در یکی از آنها باشد.

///

## ابتدا آموزش را بخوانید

بخش‌های بعدی فرض می‌کنند که شما قبلاً [آموزش - راهنمای کاربر: امنیت](../../tutorial/security/index.md){.internal-link target=_blank} اصلی را خوانده‌اید.

همه آنها بر اساس مفاهیم مشابه هستند، اما برخی قابلیت‌های اضافی را فراهم می‌کنند.
