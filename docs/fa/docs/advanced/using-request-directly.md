# استفاده مستقیم از Request

تا کنون، شما بخش‌هایی از درخواست را که نیاز دارید با تایپ‌هایشان تعریف کرده‌اید.

گرفتن داده‌ها از:

* مسیر به عنوان پارامترها.
* هدرها.
* کوکی‌ها.
* و غیره.

و با انجام این کار، **FastAPI** آن داده‌ها را اعتبارسنجی می‌کند، تبدیل می‌کند و مستندات API شما را به طور خودکار تولید می‌کند.

اما مواقعی وجود دارد که ممکن است نیاز به دسترسی مستقیم به شیء `Request` داشته باشید.

## جزئیات درباره شیء `Request`

از آنجا که **FastAPI** در واقع **Starlette** در زیر آن است، با لایه‌ای از ابزارهای متعدد در بالای آن، می‌توانید هنگام نیاز مستقیماً از شیء <a href="https://www.starlette.io/requests/" class="external-link" target="_blank">`Request`</a> Starlette استفاده کنید.

این همچنین به این معناست که اگر داده‌ها را مستقیماً از شیء `Request` بگیرید (مثلاً بدنه را بخوانید)، توسط FastAPI اعتبارسنجی، تبدیل یا مستند نخواهد شد (با OpenAPI، برای رابط کاربری خودکار API).

اگرچه هر پارامتر دیگری که به طور عادی تعریف شده باشد (مثلاً بدنه با مدل Pydantic) همچنان اعتبارسنجی، تبدیل، حاشیه‌نویسی و غیره خواهد شد.

اما موارد خاصی وجود دارد که دریافت شیء `Request` مفید است.

## استفاده مستقیم از شیء `Request`

تصور کنید می‌خواهید آدرس IP/هاست کلاینت را داخل *تابع عملیات مسیر* خود دریافت کنید.

برای این کار باید مستقیماً به درخواست دسترسی داشته باشید.

{* ../../docs_src/using_request_directly/tutorial001.py hl[1,7:8] *}

با تعریف یک پارامتر *تابع عملیات مسیر* با تایپ `Request`، **FastAPI** می‌داند که `Request` را در آن پارامتر ارسال کند.

/// tip

توجه داشته باشید که در این مورد، ما یک پارامتر مسیر را در کنار پارامتر درخواست تعریف می‌کنیم.

بنابراین، پارامتر مسیر استخراج، اعتبارسنجی، تبدیل به تایپ مشخص شده و حاشیه‌نویسی با OpenAPI خواهد شد.

به همین ترتیب، می‌توانید هر پارامتر دیگری را مانند حالت عادی تعریف کنید و علاوه بر آن، `Request` را نیز دریافت کنید.

///

## مستندات `Request`

می‌توانید جزئیات بیشتری درباره <a href="https://www.starlette.io/requests/" class="external-link" target="_blank">شیء `Request` در سایت مستندات رسمی Starlette</a> بخوانید.

/// note | جزئیات فنی

شما همچنین می‌توانید از `from starlette.requests import Request` استفاده کنید.

**FastAPI** آن را مستقیماً فقط به عنوان یک سهولت برای شما به عنوان برنامه‌نویس فراهم می‌کند. اما مستقیماً از Starlette می‌آید.
