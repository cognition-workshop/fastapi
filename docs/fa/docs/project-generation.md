# قالب Full Stack FastAPI

قالب‌ها، در حالی که معمولاً با یک تنظیمات خاص ارائه می‌شوند، طوری طراحی شده‌اند که انعطاف‌پذیر و قابل سفارشی‌سازی باشند. این به شما امکان می‌دهد آنها را بر اساس نیازهای پروژه خود تغییر داده و تطبیق دهید، که آنها را به یک نقطه شروع عالی تبدیل می‌کند. 🏁

شما می‌توانید از این قالب برای شروع استفاده کنید، زیرا شامل بسیاری از تنظیمات اولیه، امنیت، پایگاه داده و برخی endpoint‌های API است که از قبل برای شما انجام شده است.

مخزن GitHub: <a href="https://github.com/tiangolo/full-stack-fastapi-template" class="external-link" target="_blank">قالب Full Stack FastAPI</a>

## قالب Full Stack FastAPI - پشته فناوری و ویژگی‌ها

- ⚡ [**FastAPI**](https://fastapi.tiangolo.com) برای API بک‌اند پایتون.
    - 🧰 [SQLModel](https://sqlmodel.tiangolo.com) برای تعاملات پایگاه داده SQL پایتون (ORM).
    - 🔍 [Pydantic](https://docs.pydantic.dev)، استفاده شده توسط FastAPI، برای اعتبارسنجی داده‌ها و مدیریت تنظیمات.
    - 💾 [PostgreSQL](https://www.postgresql.org) به عنوان پایگاه داده SQL.
- 🚀 [React](https://react.dev) برای فرانت‌اند.
    - 💃 استفاده از TypeScript، هوک‌ها، [Vite](https://vitejs.dev) و سایر اجزای یک پشته فرانت‌اند مدرن.
    - 🎨 [Chakra UI](https://chakra-ui.com) برای کامپوننت‌های فرانت‌اند.
    - 🤖 یک کلاینت فرانت‌اند تولید شده به صورت خودکار.
    - 🧪 [Playwright](https://playwright.dev) برای تست End-to-End.
    - 🦇 پشتیبانی از حالت تاریک.
- 🐋 [Docker Compose](https://www.docker.com) برای توسعه و تولید.
- 🔒 هش رمز عبور امن به صورت پیش‌فرض.
- 🔑 احراز هویت توکن JWT.
- 📫 بازیابی رمز عبور مبتنی بر ایمیل.
- ✅ تست‌ها با [Pytest](https://pytest.org).
- 📞 [Traefik](https://traefik.io) به عنوان پروکسی معکوس / متعادل‌کننده بار.
- 🚢 دستورالعمل‌های استقرار با استفاده از Docker Compose، شامل نحوه تنظیم یک پروکسی Traefik فرانت‌اند برای مدیریت گواهی‌های HTTPS خودکار.
- 🏭 CI (یکپارچه‌سازی مداوم) و CD (استقرار مداوم) بر اساس GitHub Actions.
