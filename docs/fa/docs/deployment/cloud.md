# استقرار FastAPI روی ارائه‌دهندگان ابری

شما می‌توانید عملاً از **هر ارائه‌دهنده ابری** برای استقرار برنامه FastAPI خود استفاده کنید.

در بیشتر موارد، ارائه‌دهندگان ابری اصلی راهنماهایی برای استقرار FastAPI با آنها دارند.

## ارائه‌دهندگان ابری - اسپانسرها

برخی از ارائه‌دهندگان ابری ✨ [**اسپانسر FastAPI**](../help-fastapi.md#sponsor-the-author){.internal-link target=_blank} ✨ هستند، این امر **توسعه** مداوم و سالم FastAPI و **اکوسیستم** آن را تضمین می‌کند.

و این نشان‌دهنده تعهد واقعی آنها به FastAPI و **جامعه** آن (شما) است، زیرا آنها نه تنها می‌خواهند **خدمات خوبی** به شما ارائه دهند بلکه همچنین می‌خواهند مطمئن شوند که شما یک **فریم‌ورک خوب و سالم** دارید، FastAPI. 🙇
